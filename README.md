## 核心架构原则

| 层级 | 职责 | 核心公式 |
|------|------|----------|
| **AI层** | 识别概率优势 | signal = P_up - P_down |
| **库存层** | 控制仓位安全 | 目标仓位 = 基础仓位 + 信号强度 × 调整仓位 |
| **风控层** | 拥有最高权限 | 风控 > 库存 > AI |

**重要：风控系统永远比AI模型权限更高。模型说可以买，风控说不能买，那就不能买。**

## 合规边界（第一版）

- **T+1约束**：今日买入不可卖，区分昨日可卖库存和今日新增库存
- **高频认定**：单账户每秒300笔以上/单日20,000笔以上进入高频认定，第一版主动限频
- **印花税**：卖出端减半征收（2023年8月28日起）
